# Prerequisites

## Overview

This guide installs all required packages for building and running Nagios Core on Ubuntu.

## Install Required Packages

Install all required dependencies for:

- Nagios Core
- Nagios Plugins
- PHP-FPM
- FastCGI
- Build Tools

```bash
sudo apt-get install -y \
autoconf \
gcc \
libc6 \
make \
wget \
unzip \
libgd-dev \
python3-pip \
python3-dev \
apache2-utils \
php8.3 \
php8.3-fpm \
```
spawn-fcgi \
fcgiwrap
