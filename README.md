# Nagios Monitoring Server

## Overview

This project demonstrates the installation and configuration of Nagios Core from source on Ubuntu.

The monitoring server uses:

- Nagios Core 4.5.10
- Nagios Plugins 2.4.12
- NRPE
- PHP-FPM
- Nginx
- fcgiwrap
- Custom Monitoring Plugins
- Basic Authentication

The project documents the complete installation process, configuration, troubleshooting, and verification steps.
